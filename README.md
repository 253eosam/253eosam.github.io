# 개인 학습 노트

Claude AI와 대화하며 공부한 내용을 체계적으로 정리하는 개인 학습 노트 저장소다.

## 구조

```
content/
├── in-progress/     # 작성 중인 노트
└── completed/       # 완성된 노트
```

- **노트 폴더**: `YYYY-MM-DD-토픽명/note.md`
- **가이드라인**: `CLAUDE.md`

## 워크플로우

1. `content/in-progress/YYYY-MM-DD-토픽명/note.md` 생성
2. Claude AI와 대화하며 내용 정리
3. 완성 후 `content/completed/`로 이동
